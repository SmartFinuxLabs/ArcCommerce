# SUC-002: Create Manual Supplier Invoice Draft

Parent use case: UC-002  
Target sprint: Sprint 1.1  
Priority: P0

## Objective

Allow supplier to create structured invoice data without uploading a PDF.

## Steps

1. Supplier opens create invoice form.
2. Supplier selects buyer.
3. Supplier enters invoice number, amount, currency, issue date, maturity date, PO reference, delivery reference, and line items.
4. Platform validates required fields.
5. Supplier saves draft or submits.

## Acceptance Criteria

- PDF upload is not required.
- Draft invoice can be saved.
- Required fields are validated before submission.
